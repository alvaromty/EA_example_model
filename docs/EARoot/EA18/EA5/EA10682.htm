This is an example of generating code for Activity diagram with multiple loops.<br/>    It demonstrates the ability to dynamically identify the loop nodes.<br/>    Support multiple targeted languages such as Java, C#, C++, OOC, VBNet<br/><br/><ul>
<li><b>Steps to run:</b></li></ul>
        1. Context menu on "TestS3electionSort" | Generate Code | Specify the generating Path E.g. "C:\foo\TestSelectionSort.java"<br/>        2. If your JDK, JAVA_HOME, CLASSPATH were set properly, you can simply test the generated code like this:<br/>            C:\foo#gt;javac TestSelectionSort.java<br/>            C:\foo#gt;java TestSelectionSort<br/><br/><ul>
<li>The <b>generated code</b> for Activity "SelectionSort" would look like this:</li></ul>
<br/> public void SelectionSort()<br/> {<br/> // behavior is a Activity<br/> int i = 0, j = 0, min = -1, temp = -1;<br/> while (i #lt; N - 1)<br/> {<br/> min = i;<br/> j = i+1;<br/> while (j #lt; N)<br/> {<br/> if (a[j] #lt; a[min])<br/> {<br/> min = j;<br/> }<br/> j++;<br/> }<br/> temp = a[min];<br/> a[min] = a[i];<br/> a[i] = temp;;<br/> i++;<br/> }<br/> }<br/><br/><ul>

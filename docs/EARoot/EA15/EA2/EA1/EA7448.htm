Create your ArcGIS UML model by dropping elements from the ArcGIS Toolbox pages onto an ArcGIS diagram such as this.<br/><br/>The 'Features' Package contains your Feature Classes, while the 'Domains' Package contains your Coded Value Domains and Range Domains.<br/><br/><b>Before exporting your model to an ArcGIS schema, you must define at least one Spatial Reference element.</b> Spatial Reference elements are referred to by other schema elements via a dynamically linked Tagged Value, named 'SpatialReference'.<br/><br/>A sample Spatial Reference element has been created for you under the 'Spatial References' Package.  To add a valid coordinate system and default values to the element, simply select the element 'MySpatialReference' in the Browser window or on a diagram and then from the 'Publish' ribbon, select the option: <b>'Technologies #gt; ArcGIS #gt; Set Coordinate System'.</b><br/><br/>To export your model as an ArcGIS schema, select the ArcGIS Workspace in the Browser window, then right-click and choose:<br/><b>'Specialize #gt; Technologies #gt; ArcGIS #gt; Export to ArcGIS Workspace XML'</b><br/><br/>

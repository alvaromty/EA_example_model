The contents of an «ApplicationSchema» package will be exported as a GML Application Schema file.<br/>The tags <b><i>xmlns</i></b>, <b><i>targetNamespace </i></b>and <b><i>xsdDocument </i></b>are mandatory for an «ApplicationSchema» package<br/>You can fill in these tags either using: <br/><ul>
<li><b>Tagged Values </b>window  ( menu: View -#gt; Tagged Values ) ( or )</li><li><b>Generate GML Application Schema dialog</b> ( menu: Extensions -#gt; GML -#gt; Generate GML Application Schema )</li></ul>
To generate a GML Application Schema:<br/><ul>
<li>Select the «ApplicationSchema» Package in the Browser window</li><li>From the 'Publish' ribbon, select the option:  'Extensions -#gt; GML -#gt; Generate GML Application Schema'</li></ul><p/>

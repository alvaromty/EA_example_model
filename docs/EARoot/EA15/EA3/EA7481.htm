Sparx System's MDG Technology for GML is the implementation of the Open Geospatial Consortium's Geography Markup Language (GML) 3.3 for Enterprise Architect.<br/>It provides an XML grammar for geographical feature modeling capabilities within Enterprise Architect.<br/><br/>Through the MDG Technology for GML, you can:<br/><ul>
<li>Apply a UML Profile for the Geography Markup Language (GML) 3.3</li><li>Make use of customized diagram types and toolbox pages, for convenient access to elements and relationships to model geographical features effectively</li><li>Generate GML Application Schema file</li></ul><p/>

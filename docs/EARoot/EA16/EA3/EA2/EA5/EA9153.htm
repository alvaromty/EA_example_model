This parametric diagram describes the 'Echo' audio effect used by the DSP.<br/><ol>
<li>The original sound source is created by the Constraint Block instance of <i>:SineWave</i>.</li><li>The echo signal is then generated by scaling the original signal through <i>:Mult</i> and offsetting it by a prescribed amount of time steps in <i>:Delay</i>.</li><li>Both the original signal and the echo signal are combined in <i>:Add2</i> to create the actual output signal.</li></ol>
For more information on particular constraints, inputs or outputs, please consult their corresponding element notes.<br/><br/>To simulate this parametric model right-click on the diagram and select the <b>SysML | Simulate Diagram... </b>command<br/><br/>Example input parameter values:<br/><ul>
<li><b><u>amplitude:</u></b> 5 (eg 5 decibels)</li><li><b><u>f:</u></b> 0.5 (eg 2 cycles per second = 2 hertz)</li><li><b><u>t:</u></b> 0 - 10 [0.05] (eg Simulate from 0 - 10 seconds calculating every 0.05 seconds)</li><li><b><u>att:</u></b> 0.5 (eg echo signal will be half of the volume of the original signal)</li><li><b><u>offset:</u></b> 1 (eg use the original signal<i> </i>value from 1 timestep ago to calculate the echo signal).</li></ul><p/>

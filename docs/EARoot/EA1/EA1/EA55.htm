The Relationship Matrix is a spreadsheet display of relationships between model elements within a Package or between elements in two different Packages.<br/>The matrix shows all the relationships of a specified type between source and target elements.  It lists the Source Package elements down the side of the matrix and the Target Package elements across the top of the matrix.  A relationship between two elements is indicated by an arrow where the relevant row and columns intersect, pointing in the direction of the relationship.<br/><br/>The Relationship Matrix can be accessed from the 'Design' ribbon by choosing the option:  <font color="#00008b">'Impact #gt; Matrix'</font>.<br/><br/>To see an example of a Relationship Matrix, once open, load the saved profile 'Requirements Trace'.<br/><br/><br/><br/>

Enterprise Architect allows modelers to record a range of information associated with model elements, to assist Project Managers in managing the project.  Information such as the Effort required to complete work on an element, the Resources allocated to work on an element, Risks associated with an element, and properties such as the Status of the element.<br/>There is a broad range of facilities available for viewing and updating the various Project Management information items.<br/>Enterprise Architect also provides a message post and reply facility that allows teams to discuss issues relating to an element, such as design decisions or risk mitigation strategies, with the discussion thread being saved as part of the element information.<br/>Click on these buttons to see examples of how to use the various Project Management tools available in Enterprise Architect.<br/><br/>

Appears In: Heat Maps
A Heat Map is a chart that uses a grid-based layout, where the color, size and grouping of its cells, represent specific aspects of a data set.  Heat Maps are typically used to illustrate co-dependant properties that could influence strategic or project level decisions.<br/><br/>Enterprise Architect supports a number of ways of the retrieving data used to create Heat Maps.  In the examples presented here, the data is defined using CSV lists.<br/><br/>To view the data source for a chart, open its Properties window.<br/>

This Class diagram uses Association connectors to show the relationships between stakeholders. A number of the connectors have a Tagged Value (_bezier=true) assigned to allow them to be bent in a curve. The Associations have been gives names that indicate the nature of the relationship and the arrows indicate the direction of the relationship.<br/>

A number of tools are available for updating diagrams to show traceability:<br/><ul>
<li>On a diagram, right-click an element and choose:  'Insert Related Elements' to automatically add elements that are linked to the selected element</li><li>On the Relationships window, use the context menu option 'Place Related Element in Diagram' to add elements to the diagram that are directly linked to the selected element</li><li>On the Traceability Window, add elements that are indirectly linked</li></ul>

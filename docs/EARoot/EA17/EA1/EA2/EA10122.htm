This example makes use of sample source code that is part of your Enterprise Architect installation.<br/>The sample code for this example is installed to:  "C:\Program Files (x86)\Sparx Systems\EA\Code Samples\VEA"<br/>but should be copied to a different location, so that you don't need to run with elevated permissions in order to compile and build the project.<br/>So, for example, copy everything in "C:\Program Files (x86)\Sparx Systems\EA\Code Samples\VEA", <br/>to "C:\EA_CodeSamples\VEA".<br/><br/>Before running the example, you must configure Local Path definitions for %VEA% and %MSVS%.<br/>-    %VEA% = "C:\EA_CodeSamples\VEA"    (the directory into which you copied the sample code)<br/>-    %MSVS% = "C:\program files (x86)\Microsoft Visual Studio 9.0\Common7\IDE"<br/>(From the 'Develop' ribbon, select the option 'Preferences #gt; Options #gt; Configure Local Paths'.)<br/><br/>1.  Open the Debugger Window.  It can be accessed from the 'Execute' ribbon, by selecting the option 'Tools #gt; Debugger #gt; Open Debugger'.<br/>2.  Open the Execution Analyzer window.  It can be accessed from the 'Execute' ribbon, by selecting the option 'Tools #gt; Analyzer'.<br/>3.  Locate the Analyzer script 'Book Test'.  It can be found in the Execution Analyzer window under 'Visual Execution Analysis<b>..</b>Net<b>.</b>Source'.<br/>4.  Select the script (by placing a checkmark), then build the project by clicking on the 'Build' button.<br/>5.  View the source code for class Test.  Select class Test on this diagram, then right-click and choose 'Code Engineering | View Source Code'.<br/>6.  Create a breakpoint in the source code at, say, line 89 (an assignment to 'priceBooks').<br/>7.  In the Debug window, click on the 'Run' button.<br/>8.  The program will run, but should pause when it reaches the breakpoint.<br/>9.  The execution marker should be highlighting line 89.<br/>10.  Hover the mouse cursor over various identifiers in the source code - the variable's value will be displayed.<br/>11.  Click 'Run' again to continue execution.<br/><br/>

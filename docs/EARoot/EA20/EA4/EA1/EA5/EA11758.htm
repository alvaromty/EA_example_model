<b>Modeling Tips</b><br/>This BKM defines 3 parameters:<br/><ul>
<li>Rate : number</li><li>Term : number</li><li>Principle : number</li></ul>
<br/>The function body is implemented as Literal Expression<br/><ul>
<li>calculates the payment for a loan based on constant payments and a constant interest rate</li></ul>
<br/><b>Example</b><br/>Given a home loan with amount of 300000 for 30 years, at the annual interest rate of 4%; What is the MONTHLY repayment?<br/><br/>Provide the "Values for Simulation Parameters" as follows:<br/><ul>
<li>Rate: 0.04 / 12</li><li>Term: 30 * 12</li><li>Principle: 300000</li></ul>
<br/>Validate the model and run the simulation. The result is 1432.25 (NOTE: the expression used built-in function "decimal" to format the floating result.)<br/><br/>The runtime simulation results will be showing on in the Literal Expression.<br/>

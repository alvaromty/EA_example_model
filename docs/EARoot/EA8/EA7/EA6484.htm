Appears In: The Gantt Views
The Gantt View is a tool that charts the allocation of resources to elements in a project, Package or diagram.  <br/>There are a number of different Gantt charts available:<br/><ul>
<li>The Project Gantt View  - used to view elements across the entire repository</li><li>Personal Gantt View  - used to view task assignments for the current user, based on resource allocations</li><li>Diagram Gantt View  - used to display the allocation of resources to the elements in a given diagram</li><li>Package Gantt View  - used to display the allocation of resources to the elements contained in a selected Package</li></ul>

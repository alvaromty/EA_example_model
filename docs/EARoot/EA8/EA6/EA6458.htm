Appears In: Element Discussions
Enterprise Architect provides facilities for attaching any number of discussion threads to elements within your model.<br/>The Collaborate window allows you to create, read and reply to discussions for the selected element, whilst the Discussions window lists all discussions, with various options for sorting and filtering.<br/>Using the 'Discuss' tab of the dockable Collaborate window, you can review and participate in discussions on different elements simply by clicking on each element in turn; the Collaborate window immediately switches to the discussion of the new selected element.<br/>The Discussions window, on the other hand, lists all discussions on all elements within the model, with options for sorting and filtering the display.  This window is typically used for managing discussion threads; for example, removing discussions that are no longer needed.  There is also an option to edit (contribute to) the selected discussion thread.<br/>Both windows can be accessed through the 'Start #gt; Collaborate #gt; Discussions' ribbon option.<br/><br/><br/><br/><br/>

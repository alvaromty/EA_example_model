To display the Calendar, from the 'Start' ribbon, select the option: 'Collaborate #gt; Calendar'.<br/>The Calendar can be used to schedule and view events such as meetings, milestones, reviews and workshops. It can also be used to view Project Tasks or to view the allocation of project resources to model elements.<br/>To switch between views, use the drop-down list and select Project Calendar, Allocated Resources or Project Tasks.<br/>Double-clicking an item in the Calendar displays the details for that item.  <br/>By right-clicking on a Resource Allocation, you can locate the associated model element in the Browser window.<br/>

Appears In: Allocating Resources
Resource Allocation is performed using either the Resource Allocation window or the 'Assigned Resources' dialog.<br/>Select an element and then, from the 'Construct' ribbon, select the option 'Task Management #gt; Resource Allocation', or press Ctrl+Shift+7.  <br/>Alternatively, you can select an element on a Gantt chart, right-click and choose 'Assign Resource'.<br/><br/>

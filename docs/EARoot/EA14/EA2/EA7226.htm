National Information Exchange Model (NIEM) provides a common framework that is used to define how information can be shared between systems, government agencies and organizations. <br/><br/>Enterprise Architect provides an MDG Technology for NIEM, that helps you to:<br/><ul>
<li>Create and develop UML-based Information Exchange Package Documentation (IEPD) models, by providing starter models, model Patterns and a number of toolboxes for creating IEPD models and schema models</li><li>Generate complete IEPDs from your IEPD model</li><li>Generate NIEM conformant schemas from your information models</li><li>Import NIEM Reference Schema into your model</li><li>Create NIEM subset namespaces, composed from elements of the NIEM Reference Schemas</li><li>Create PIM, PSM and Model Package Description (MPD) diagrams, using the NIEM Toolbox pages</li></ul><p/>

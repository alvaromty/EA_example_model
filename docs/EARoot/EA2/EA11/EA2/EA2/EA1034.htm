EA_OnOutputItemClicked events inform Add-Ins that the user has clicked on a list entry in the system tab or one of the user defined output tabs. <br/>Usually an Add-In responds to this event in order to capture activity on an output tab they had previously created through a call to Repository.AddTab(). <br/>Note that every loaded Add-In receives this event for every click on an output tab in Enterprise Architect, irrespective of whether the Add-In created that tab. Add-Ins should therefore check the TabName parameter supplied by this event to ensure that they are not responding to other Add-Ins' events.<br/><br/><b>Return Value</b><br/>None.<br/><br/><b>User Guide</b><br/><a href="http://www.sparxsystems.com/enterprise_architect_user_guide/15.2/">EA_OnOutputItemClicked</a><br/>

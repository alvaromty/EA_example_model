This diagram uses the firewall profile to demonstrate how the docked Properties window now shows profile properties for custom metamodels above the properties that UML defines for the Classes that they are extending.<br/>The Properties Window can be accessed using the ribbon:<br/><ul>
<li>Start | Desktop | Design | Properties</li></ul>
<br/>Or the keyboard accelerators:<br/><ul>
<li>'Ctrl + 2'</li><li>'Alt + 1'  #gt; Properties</li></ul>
<br/>Try selecting the different elements and connectors listed.<br/>Try changing different properties.<br/>Try collapsing the General, and Class and Association property groups to simplify the options view.<br/>

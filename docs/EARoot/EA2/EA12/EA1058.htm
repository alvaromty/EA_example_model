Appears In: Element Browser and Properties Window
As of EA 15.2, the Element Browser has become the 'Details' tab of the Inspector window.<br/>When used in conjunction with the docked Properties window, the 'Details' tab provides quick and easy access to the details of any and all of an element's properties.<br/>As an example, run through these steps:<br/><ul>
<li>Ensure that the docked Properties window is showing (ribbon: 'Start #gt; Desktop #gt; Design #gt; Properties')</li><li>Ensure that the 'Details' tab is showing (ribbon: 'Start #gt; Desktop #gt; Design #gt; Inspector #gt; Details')</li><li>On the diagram, select 'Class1'</li><li>Right-click within the 'Details' tab and choose 'Navigation Options'; select all node types, then click on the OK button</li><li>Click within the 'Details' tab, then press (<b>Ctrl  '+ '</b>) to expand all nodes</li><li>Click on the various nodes within the 'Details' tab and observe that the details for the selected item are displayed in the docked Properties window</li><li>Select a different element on the diagram; the contents of the 'Details' tab are updated accordingly, allowing you to inspect properties of this element's features</li></ul><p/>

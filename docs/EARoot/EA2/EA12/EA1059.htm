'Navigation Options' allow you to specify node types within the Element Browser, that you wish to show or hide.  As of EA 15, the Navigation Options can now be saved as named sets, allowing you to switch easily from one set to another.<br/><br/>To see an example, perform the following steps:<br/><ul>
<li>Select 'Class1' in the diagram</li><li>Display the Element Browser</li><li>Right-click in the Element Browser, then choose 'Navigation Options'</li><li>In the 'Select Visible' dialog, select 'Operations', 'Attributes' and 'Receptions' and deselect all others.  </li><li>Save this selection as 'Atts and Ops'.</li><li>Now select 'Requirements', 'Constraints' and 'Scenarios' and deselect all others.</li><li>Save this selection as 'Reqs'.</li><li>Click on OK to dismiss the 'Select Visible' dialog.</li></ul>
<br/>You can now right-click in the Element Browser and select 'Atts and Ops' to show only the nodes from that set, or choose 'Reqs' to show only the nodes in that set.<br/>

Appears In: Elements' Compartment Visibility
The elements on this diagram have had features added to them, to provide examples of each of the different types of element compartment that can be displayed.<br/>Visibility of the element compartments can be set at the diagram level, as well as visibility of additional compartments being set for individual elements on the diagram.<br/>To set diagram level compartment visibility:<br/>Double-click on the background of the diagram to open the diagram's Properties dialog, then select the 'Elements' page.  At the center of this page is a large checklist that allows you to enable and disable the display of each compartment.  This affects all elements on the diagram.<br/>To enable compartment visibility for an individual element:<br/>Right-click on a diagram element, then choose 'Compartment Visibility' from the pop-up menu.  In the 'Select Visibility' dialog, select any additional compartments that you want to display for this element.<br/>Note:  Selecting a compartment in this checklist will show that compartment for this element, however, deselecting compartments here will <b>not </b>hide a compartment that is made visible at the diagram level.<br/>

Appears In: Dynamic Charts
DynamicChart artifacts are Chart elements that are styled and rendered dynamically when the diagram opens. <br/>They rely completely on code to define their series, style and content and are managed entirely through the Automation Interface by clients - typically scripts or plug-ins.<br/><br/>JavaScript is the primary language used to define Dynamic Charts, but it is also possible to call functions written in other languages.<br/><br/>This Package contains examples of many charts, each described dynamically by JavaScript clients. <br/>Each chart example presents the two methods of rendering - a JavaScript coding method and a JSON datasource method.<br/><br/>Each DynamicChart artifact has a user defined JavaScript function, ConstructChart( ), that is automatically invoked whenever a diagram containing the artifact is opened.  That function is responsible for defining the style and populating the data for the chart that is displayed.<br/><br/>Using a JSON data source involves two fundamental steps:<br/><ul>
<li>first, create a JSON variable that defines the styles and data of the chart to render,</li><li>then call the built-in function ConstructChartFromJSON, passing 'elemGUID' as the first parameter and the JSON structure as the second argument.</li></ul><p/>

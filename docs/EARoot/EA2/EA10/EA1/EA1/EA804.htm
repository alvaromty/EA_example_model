An axis can be split to allow another series to display other information side by side.<br/><br/>In BCG this achieved through calling Split on one axis and then calling SetRelatedAxis on the series passing it the result of the Split function.<br/><br/>In the Stock example, the Y axis is split. A new volume series is then created and given the child axis. The option allows multiple series to share the same axis 'space'<br/>

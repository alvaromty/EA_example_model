Appears In: Custom Drawing Styles
When the option 'Custom Style' is applied to a diagram (found in the 'Appearance' section of the diagram's Properties window) or to an element (found under the element's context menu | Appearance | Enable Custom Draw Style), a range of formatting buttons will be presented above the context menu when you right-click an element on the diagram.<br/>The buttons allow you to set these properties on the selected element:<br/><ul>
<li>shape</li><li>colors</li><li>opacity</li><li>text alignment</li><li>text rotation</li><li>icon position (when applied)</li><li>icon size (when applied)</li><li>border style</li><li>stack</li><li>stack direction</li></ul><p/>

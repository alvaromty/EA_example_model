The Model Search facility can be used to find anything that exists in the repository, locating information easily no matter where it is in the model.<br/>The facility includes a wide range of built-in searches and users can also define their own searches using the Query Builder or SQL Builder.<br/>Objects can be located by searching for the name of an element, text in element notes, a Tagged Value or by searching for specific properties such as when an element was created or who created it.<br/>The result set can be sorted, individual elements can be located in the Browser window.<br/>Search queries can also be saved and used as input for other tools, such as Model Views and the Document Generator.<br/><br/>The Model Search facility can be accessed from any ribbon, by selecting the option:<br/>'Explore #gt; Search #gt; Model'<br/>It can also be accessed from the 'Design' ribbon, by selecting the option:<br/>'Element #gt; Manage #gt; Search Model'.<br/>

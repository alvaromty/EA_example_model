Appears In: Element Appearance
Changing the appearance of an element on a diagram, or a selection of elements, can provide a useful visual cue to highlight a particular business meaning.<br/>While all elements in the repository have a global default appearance, the default can be overridden for individual elements either globally or for specific diagrams.  The Fill Color, Border Color, Line Thickness, Font Style and Font Color can all be changed, as global defaults for all elements, as global settings for specific elements, or as local settings for a specific element on a particular diagram.<br/>It is also possible to assign a Shape Script or an alternate image to a stereotype and apply that stereotype to particular elements, as well as assigning an alternate image to a specific element.<br/><br/>To configure the default appearance for all elements, from the 'Start' ribbon, select the option:<br/>'Workspace #gt; Preferences #gt; Standard Colors, Appearance'<br/><br/>To configure the global appearance of a specific element, right-click on the element on a diagram, then choose:  'Appearance | Default Appearance'<br/>To configure the local appearance of a specific diagram object, select the object on a diagram and using the 'Layout' ribbon, select options from the 'Style' panel to set Font Style, Font Color, Fill Color, Line Color or Line Thickness.<br/>These elements illustrate some of the options available for setting element appearance.<br/><br/>

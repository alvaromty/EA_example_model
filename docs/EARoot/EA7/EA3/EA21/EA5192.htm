To show the Browser window, select the ribbon option:  'Start #gt;Desktop #gt; Design #gt; Browser'.<br/>The Browser window is the primary tool for structuring and organizing the content of your project repository.<br/>It presents the model as a hierarchical tree structure, consisting of Packages, diagrams and elements.<br/>Packages can contain diagrams, elements and other Packages.  Elements can contain attributes and operations, as well as diagrams and other elements.<br/><br/>The key structural element is the Package, which is a folder-like element that can contain diagrams, elements and other Packages.<br/><br/>With the Browser window, you can easily review the model structure and if required, modify it by simply dragging items from one location to another.  Tree nodes including Packages, diagrams, elements and Features (the attributes and operations of an element) can also be copied and pasted between locations.<br/><br/>Context menus for each of the different node types within the Browser window provide a wide range of commands for working with each of the different node types.<br/><br/>Typically, the Browser window would be docked on one side or the other of your Enterprise Architect workspace, but it can also be 'floated' to any position on your desktop workspace, or hidden altogether.<br/>

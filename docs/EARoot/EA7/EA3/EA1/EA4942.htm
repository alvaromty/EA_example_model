The base Artifact element provides a means for connecting elements in the repository with external files or web resources. <br/>When a file or resource is dragged onto a diagram, a menu will prompt the user to select the type of Artifact to create:<br/><ul>
<li>'Hyperlink' creates a Hyperlink element on the diagram; you can select a sub-option to define what happens when double-clicking on the Hyperlink: 'Open' displays the file content and 'Edit' opens it within the assigned file editor</li><li>'External' creates an Artifact element on the diagram that stores a hyperlink to the external file; press F12 or Ctrl+E to open the external file</li><li>'Internal' creates an Artifact element on the diagram and makes a copy of the file inside the repository; double-clicking the Artifact launches the internal document</li><li>'Insert' (graphics files only) inserts the file into the diagram as a filled Boundary element</li></ul>
<b><br/></b><b><br/></b>You can also create Artifact elements by dragging one of the 'Artifact' icons from the 'Artifacts' page of the Diagram Toolbox.<br/><br/>

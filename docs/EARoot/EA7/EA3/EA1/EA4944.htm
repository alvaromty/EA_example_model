The 'Artifacts' tool box page offers a range of other Artifact types that can be created and placed on a diagram.  Double-clicking the Artifact on the diagram will typically open or run the associated specification.<br/>They include:<br/><ul>
<li>Matrix Specification - encapsulates a saved Relationship Matrix profile</li><li>Report Specification - encapsulates a saved Document Generation specification</li><li>Model View - represents a table of model data, retrieved using a custom SQL search</li><li>Standard Chart - an artifact that records information for generating a bar chart or pie chart from data in your model; once you have defined the chart, it automatically displays with the latest information whenever you open the parent diagram</li><li>Time Series Chart - an artifact that records information for generating a linear graph of model properties over time; once you have defined the chart, it automatically displays with the latest information whenever you open the parent diagram</li><li>Executable State Machine - an artifact that specifies the objects that will be involved in a simulation</li><li>SysMLSim Configuration - records a specific SysML simulation configuration</li><li>Image Asset - an artifact used to store an image in the model repository</li><li>Reading List - provides an ordered list of elements; information from each element is displayed in the 'Dynamic Document' view</li><li>Checklist - facilitates creation of a list of items, each with a checkbox</li><li>User Story - provides a Linked Document, for recording a Business Use Case</li><li>Working Set - a saved profile that specifies a workspace layout</li></ul><p/>

The Model Views facility allows the modeler to create a number of views that can be used to group elements and diagrams in whatever combination the modeler requires.<br/>There are options to create 'Favorites' folders, 'Slide-Show' folders and Search-based folders.<br/>The 'Favorites' folder is useful to keep track of hand picked elements and diagrams of interest. Elements, diagrams and Packages are simply dragged from the Browser window and dropped into a 'Favorites' folder.<br/>The 'Slide Show' folder is useful to create a diagram slide show for the purposes of demonstrating diagrams to a group of people in a workshop or meeting.  In this case, diagrams are dragged from the Browser window and dropped into the 'Slide Show' folder.  The order of the diagrams in the folder determines the order in which they are displayed in the slide show.<br/>The 'Search' folder is useful to keep track of elements that meet specified criteria and to be notified when new elements are added to the Repository that meet the criteria. In the folder properties, you can define the search to be run that populates the folder, and the time period after which the search is re-run to refresh the contents of the folder.<br/>Model views can be created for a variety of purposes and an experienced modeler will often use the Model Views facility as the access point to the Repository. <br/>To show the 'Model Views' tab, select the ribbon option: 'Start #gt; Desktop #gt; Design #gt; Focus #gt; Views'.<br/>

Display back order as negative stock Decision Verified
Version:  		 Medium Reported: 5/4/2015 12:00:00 AM
Resolved: 5/15/2015 12:00:00 AM
Details:
The question had been raised as to how to display items that had zero items in stock but that had back orders in place. The decision has been made to display the back order quantities as a negative number, to mark them with an asterisk and to put an annotation on the report explaining the negative number. "Negative stock numbers indicate total back order quanitites"<br/>

Create table index to ensure fast retrieval of stock levels Task Verified
Version: 15056B5ED3TSK		 Medium Reported: 5/12/2015 12:00:00 AM
Resolved: 5/15/2015 12:00:00 AM
Details:
The current test run on the pre-production database shows that it is taking up to 3 seconds to return a list of stock levels for 10 selected titles using a test data set of 14K title records. When we add the application processing time and Internet latency time the user will wait for up to 7 seconds which is unacceptable. The database retrieval needs to take less than 1 second.<br/>

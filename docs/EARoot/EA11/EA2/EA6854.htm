To access the HTML Report generation facility, select the ribbon option:<br/>'Publish #gt; Model Reports #gt; HTML #gt; Standard HTML Report',  or<br/>select the Package for which to generate a report, then press Shift+F8.<br/>To run an example now:<br/><ul>
<li>Find this diagram in the Browser window, then select its parent Package 'Documentation'.</li><li>Press Shift+F8.</li><li>Click on the 'browse' button beside the 'Output to' field and navigate to a suitable folder.  Create a new folder at that location named 'Documentation'.</li><li>Now, click on the 'Generate' button, then click on the 'View' button.</li></ul>
The generated HTML report will open in your default web browser.<br/>

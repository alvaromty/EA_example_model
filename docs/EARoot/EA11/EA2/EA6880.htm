Enterprise Architect has the ability to generate an HTML report of an entire model or a single branch of the model. The HTML report provides a convenient frames-based structure navigator and content explorer with much of the output hyper-linked to related sections within the model.<br/>The generated HTML report is compatible with any standard web server - either on Unix or Windows, allowing the output to be placed within the context of your web server. <br/>

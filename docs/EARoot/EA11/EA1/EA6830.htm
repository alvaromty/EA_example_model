You can publish this diagram as a Pattern yourself and then import it as a Resource; perform these steps:<br/><ol>
<li>Select the ribbon option 'Specialize #gt; Technologies #gt; Publish-Tech #gt; Publish Diagram as Pattern'</li></ol>
In the 'Publish Diagram as Pattern' dialog, <br/><ol>
<li>Specify a distinct pattern name, such as 'myPattern'</li><li>Specify a suitable location and filename such as 'C:\temp\myPattern.xml'</li><li>Click on the OK button.</li></ol>
<br/>In the 'Resources' tab of the Browser window, <br/><ol>
<li>Right-click on the Patterns folder and select 'Import Pattern'.</li><li>Specify the filename that you used to publish the pattern.</li><li>Click on the OK button.</li></ol>
<br/>Finally, create a new Package with an empty diagram.<br/>From the 'Resources' tab of the Browser window, drag and drop 'myPattern' onto the empty diagram.<br/>Delete the Package and its contents when finished.<br/>

Appears In: XMI Import and Export
A 'Controlled Package' is a Package configured to save and load in XML format to a named file.<br/>Controlled Packages appear in the Browser Window with a small red rectangle to the left of the Package icon.<br/>The Package 'Source Pkg' is configured for Package Control.<br/>Select the Package on this diagram, then press Alt+G to <i>find </i>the Package in the Browser window.  <br/>Notice the red rectangle in the Browser window, indicating a 'Controlled Package'.<br/>Click the Package in the Browser window, then choose the 'Publish #gt; Model Exchange #gt; Package Control #gt; Package Control' ribbon option to display the 'Package Control Options' dialog.  The field 'XMI Filename' specifies the path to the file that is used to 'control' the Package.  (Path substitution can be used here, to facilitate the use of different file paths on different users' PCs.)<br/>Once a Package is configured as a 'Controlled Package', pressing Ctrl+S will save the Package to its associated XMI file and pressing 'Ctrl+L' will load the Package from the file.<br/>Batch Export or Import can also be used to Export or Import a selection of all of the model's controlled Packages in one operation.  This can be accessed by selecting the ribbon option:<br/>'Publish #gt; Model Exchange #gt; Export XML #gt; Batch Export Controlled Packages'  or<br/>'Publish #gt; Model Exchange #gt; Import XML #gt; Batch Import Controlled Packages'<br/>

The interface of the Database Builder consists of:<br/><ul>
<li>A Tree of data models, listing all defined data models in the current repository</li><li>A 'Columns' tab through which you directly manage the Table columns</li><li>A 'Constraints/Indexes' tab for the direct management of Table constraints such as Primary Keys, Foreign Keys and Indexes</li><li>An SQL Scratch Pad that you can use to run ad-hoc SQL queries against a live database</li><li>A 'Database Compare' tab that displays the results of comparisons between the data model and a live database</li><li>An 'Execute DDL' tab on which you can execute generated DDL against a live database, instantly</li></ul>
<br/>You can use the Database Builder to:<br/><ul>
<li>Create, edit and delete database objects (Tables, Views, Procedures, Sequences and Functions)</li><li>Create, edit and delete Table constraints (Primary Keys, Indexes, Unique Constraints, Check Constraints and Triggers)</li><li>Create, edit and delete Table Foreign Keys</li><li>Reverse engineer database schema information</li><li>Generate DDL from a modeled database</li><li>Compare a live database schema with a modeled database</li><li>Execute generated DDL against a live database</li><li>Execute adhoc SQL statements against a live database</li></ul><p/>

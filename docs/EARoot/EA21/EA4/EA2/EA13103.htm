Users will have a facility to define their preference for a report including delivery options. Report preferences will be able to be set at three different levels:<br/><ul>
<li>Organization</li><li>Department (Section)</li><li>Individual</li></ul>
Security will restrict access to the various levels; for some preference the precedence will be set from Organization to Individual, and for others it will be the reverse, from Individual to Organization. <br/>

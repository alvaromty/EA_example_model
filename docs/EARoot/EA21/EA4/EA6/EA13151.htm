<b>Personas User Stories</b><br/>This custom diagram shows a <b>Persona </b>linked to a <b>User Story</b>. All of the details can be seen through the <b>WebEA interface</b> which keeps all parties up-to-date and informed about progress including <b>customers</b>.<br/><br/>The Person is fully described in the <b>notes </b>for the element. <br/>The User Story shows a number of compartments which can contain extra details as the User Story is Groomed and Developed including Unit Tests. For example, requirements (responsibilities) detail the User Story in preparation for development work to begin for a particular iteration or sprint or because it has reached the top of the Queue in a project <b>Kanban board.</b><br/>

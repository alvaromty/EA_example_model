<b>Diagram Overview (OSLC) - Development Team</b><br/>This deployment diagram describes the deployment scenario of a head office with a distributed Development team that uses a different vendor's Issue Tracking software and a Testing team that uses another vendor's testing software.<br/><br/>The secure connection can be via the public Internet or through a Virtual Private Network (VPN) with an add-in communicating with the <b>Open Services for Lifecycle Collaboration </b>interface on the Pro CLoud Server..<br/><br/>Enterprise Architect can contain elements from all parts of the application lifecycle and these elements can be synchronized with representations of the elements in other tools such as Requirements Management, Issue Tracking, Testing, Configuration Management and other lifecycle tools. Enterprise Architect allows all the relationships between elements to be visualized in a wide range of diagrams, matrices and other representations.<br/><br/>

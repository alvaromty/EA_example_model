A desktop computer is a personal computer typically used at a single location such as an office desk or work area. It is commonly comprised of a box that contains circuitry and a power supply, a monitor and two input devices: a keyboard and a mouse. It can be used to launch the EA Client for model authoring or to open WebEA using a Web Browser for model viewing and collaboration.<br/>

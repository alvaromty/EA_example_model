<b>Diagram Overview - Small Team</b><br/>This deployment diagram describes he deployment scenario of a small team where one team member also does some of their work from a home office.<br/>The Enterprise Architect repository has been implemented as a file based Firebird repository which doesn't require any database administration and offers many benefits that suit the small team. The repository can be seamlessly transferred to a Relational Database as the team grows or when the need arises.<br/>

The “Yacht in Distress” Scenario

The Sample Problem applies UPDM to a common scenario in civilian maritime Search and Rescue (SAR) operations -- a Yacht in distress. A Monitor Unit picks up the distressSignal from the Yacht and passes it on to the Command and Control (C2 Center). The C2 Center coordinates the search and rescue operation among the Rescue Helo, a Naval Ship and a Rescue Boat.

This model is based on a UK MOD example model.
